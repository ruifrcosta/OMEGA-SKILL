# OMEGA Google Workspace (GWS) Automation Recipes

This document unifies and replaces all fragmented, isolated `recipe-*` scripts by establishing standardized, production-grade GWS automation recipes optimized for the OMEGA operating system.

## 1. Unified Recipe: Sheet Backup to CSV
Replaces `recipe-backup-sheet-as-csv`. Downloads a target Google Sheet tab and stores it as a localized CSV file under strict transactional safety.

### Workflow Execution
```bash
# 1. Fetch values in JSON format
gws sheets spreadsheets values get \
  --spreadsheet "SPREADSHEET_ID" \
  --range "Sheet1!A1:Z100" \
  --format csv > /backup/path/sheet_backup.csv
```

## 2. Unified Recipe: Automated Meeting Prep & Announce
Replaces `recipe-watch-drive-changes`, `recipe-create-meet-space`, `recipe-share-event-materials`, and `recipe-send-team-announcement`.
Synthesizes Calendar event creation, Google Meet space provisioning, attachment sharing via Drive, and mailing a team digest.

### Workflow Steps
```bash
# 1. Provision a Google Meet Space
gws meet spaces create --json '{"config": {"accessSettings": {"accessPolicy": "OPEN"}}}'

# 2. Schedule Calendar Event with the Meet Link and Shared Drive Documents
gws calendar events insert \
  --calendarId "primary" \
  --json '{
    "summary": "OMEGA Sprint Sync",
    "description": "Weekly alignment. Meet link: https://meet.google.com/...",
    "start": {"dateTime": "2026-05-25T10:00:00Z"},
    "end": {"dateTime": "2026-05-25T11:00:00Z"},
    "conferenceData": {"entryPoints": [{"entryPointType": "video", "uri": "https://meet.google.com/..."}]}
  }'

# 3. Email Staggered Announcement to Team
gws gmail messages send \
  --json '{"userId":"me", "message":{"raw":"From: cto@omega.io\nTo: team@omega.io\nSubject: Sprint Sync Scheduled\n\nThe weekly OMEGA sync has been scheduled. Please review materials."}}' \
  --sanitize "team-announcement-template"
```

## 3. Unified Recipe: Dynamic Mail Triage & Task Creation
Replaces `recipe-forward-labeled-emails`, `recipe-create-task-list`, and `recipe-review-overdue-tasks`.
Checks for specific unread messages, triages urgency, creates Tasks inside Google Tasks, and notifies SRE/Operations.

### Workflow Steps
```bash
# 1. Stream unread inbox items in NDJSON
gws gmail messages list --params '{"q": "is:unread label:production-alerts"}' --page-all

# 2. Append Alert Details to Google Tasks list
gws tasks tasks insert \
  --tasklist "TASKLIST_ID" \
  --json '{"title": "Production Alert: Triage Required", "notes": "Alert details streamed from Gmail."}'
```
