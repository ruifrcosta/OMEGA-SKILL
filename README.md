## 🚀 Como Utilizar

### A. Ativação via Prompt
A OMEGA é ativada de forma autônoma sempre que o agente detecta tarefas de arquitetura, infraestrutura ou design de interface:
- *"Usa a skill OMEGA para iniciar um novo projeto Next.js."*
- *"Aciona a OMEGA para fazer uma auditoria de segurança à minha API."*
- *"Desenha uma interface minimalista utilizando a OMEGA."*

### B. Vocabulário de Design (Impeccable Commands)
OMEGA implementa 23 comandos especializados de design frontend de alta fidelidade. Ative-os diretamente:
- **`craft [funcionalidade]`**: Planeia e constrói uma funcionalidade end-to-end.
- **`shape [funcionalidade]`**: Planeia a UX/UI antes de escrever qualquer linha de código.
- **`critique [alvo]`**: Revisão heurística profunda da experiência do utilizador.
- **`polish [alvo]`**: Refinamento de detalhes visuais e micro-interações antes de publicar.
- **`audit [alvo]`**: Valida acessibilidade (a11y), performance e responsividade.
- **`bolder [alvo]`** / **`quieter [alvo]`**: Torna o design mais expressivo ou mais discreto.
- **`live`**: Inicia uma sessão interativa no navegador para testar variantes visuais.

Para ver a lista completa de comandos, execute `OMEGA` sem argumentos.

### C. Utilização da OMEGA CLI
A OMEGA inclui ferramentas de linha de comando (`scripts/`) para gerir a conformidade e carregar contextos:
- **Carregar Contexto do Projeto**:
  ```bash
  node scripts/load-context.mjs
  ```
- **Inicializar o Cofre Obsidian de 33 Pastas**:
  ```bash
  python scripts/omega-cli.py init-vault --dir ./meu-projeto
  ```
- **Criar um Registro de Decisão de Arquitetura (ADR)**:
  ```bash
  python scripts/omega-cli.py create-adr --title "novo-banco-dados" --status accepted --deciders "CTO"
  ```
- **Auditar Conformidade de Segurança, CORS e Drift**:
  ```bash
  python scripts/omega-cli.py audit
  ```

## 🛡️ 绝对指令（不可妥协）
- **“无卡片”规则（视觉静音）**: 界面必须是极致极简的。严禁使用悬浮卡片容器或重度投影。完全依靠间距（4/8dp 网格）和排版来组织内容。
- **动态品牌标识**: 品牌色彩（HSL/OKLCH）和字体搭配会根据项目的目标领域动态变化（例如：医疗健康使用祖母绿，金融科技使用皇家蓝，效率工具使用罗兰紫）。
- **视觉禁用模式**: 严禁使用文本渐变色、无意义的毛玻璃特效、侧边装饰条边框，并避免将弹窗（Modal）作为第一交互方案。
- **零信任安全**: 所有数据边界输入验证、HttpOnly/Secure Cookie 以及 Argon2id 密码哈希。
