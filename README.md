# 🇵🇹 PORTUGUÊS

O **OMEGA** é o único Sistema Operacional de Engenharia de Software Autônomo e Implacável de nível Big Tech (como Stripe, Vercel, Linear). Ele unifica todas as capacidades sob uma única governança Zero-Trust e memória permanente num cofre Obsidian de 33 pastas.

## 📦 Instalação Rápida (Recomendada)

Instale a skill instantaneamente no seu ambiente **Claude Code** ou **Antigravity** executando o seguinte comando `npx`:

```bash
npx skills add ruifrcosta/OMEGA-SKILL
```

### Método de Instalação Alternativo (Git):
```bash
git clone https://github.com/ruifrcosta/OMEGA-SKILL.git ~/.gemini/config/skills/omega
```

---

## 🚀 Como Utilizar

### A. Ativação via Prompt
A OMEGA é ativada de forma autônoma sempre que o agente detecta tarefas de arquitetura, desenvolvimento ou infraestrutura:
- *"Usa a skill OMEGA para iniciar um novo projeto Next.js."*
- *"Aciona a OMEGA para fazer uma auditoria de segurança à minha API."*

### B. Utilização da OMEGA CLI
A OMEGA inclui uma poderosa ferramenta de linha de comando em Python (`scripts/omega-cli.py`) para gerir a conformidade local:

- **Inicializar o Cofre Obsidian de 33 Pastas**:
  ```bash
  python scripts/omega-cli.py init-vault --dir ./meu-projeto
  ```
- **Criar um Registro de Decisão de Arquitetura (ADR)**:
  ```bash
  python scripts/omega-cli.py create-adr --title "novo-banco-dados" --status accepted --deciders "CTO"
  ```
- **Auditar Conformidade de Segurança, CORS e Drift**:
  ```bash
  python scripts/omega-cli.py audit
  ```

---

## 🛡️ Diretivas Absolutas (Não-Negociáveis)
- **Silêncio Visual (Regra "Sem Cartões")**: As interfaces devem ser ultra-minimalistas. Proibido o uso de caixas elevadas ou sombras. A separação é feita exclusivamente por espaçamento 4/8dp e tipografia.
- **Identidade Dinâmica**: Cores e fontes adaptam-se automaticamente ao domínio do projeto (Esmeralda para Saúde, Azul para Finanças, Violeta para Produtividade).
- **Zero Trust**: Validação estrita de todas as fronteiras de dados, cookies HttpOnly/Secure e hashes de senha Argon2id.
